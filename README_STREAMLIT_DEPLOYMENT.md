# Building Project Prediction System - V9 Final

## What is fixed in this version
- Removed the extra small image from the homepage.
- Added the SPbSTU building photo as the top hero background.
- Added a Start New Project / Clear Form button after prediction.
- Removed debug Matplotlib text from the PDF report tab.
- Added English/Russian language selector. English is default.
- Added USD/RUB display selector. USD is default. RUB uses a user-entered exchange rate.
- Model calculation remains unchanged and uses the uploaded BuildingPredictor logic.
- PDF report includes green styling, university logo, and embedded graphs.

## Deploy
1. Extract this zip.
2. Upload all extracted files and the `.streamlit` folder to GitHub.
3. In Streamlit, deploy with main file path: `app.py`.
4. Reboot / redeploy the app.
